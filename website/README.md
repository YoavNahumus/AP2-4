# How to Run:
clone the repository
```bash
git clone https://github.com/YoavNahumus/AP2-2.git
```
navigate to the project directory
```bash
cd AP2-2
```
install dependencies
```bash
npm install
```
run the project
```bash
npm start
```
open [http://localhost:3000](http://localhost:3000) to view it in the browser.
